# ===

===

强类型判断相等（类型必须相同）

![img](img\===\image.png)  


***
## 输入类型
v1：any  
v2：any  


***
## 返回类型
Boolean


***
## 使用示例
![2](img\===\2.png)  


***
## 源码
```js title="/categorys/string_and_type.js"
    {//2.0.0
        opcode: `${category_id}.===`,
        messageId: '[v1] === [v2]',
        categoryId: category_id,
        type: type.BlockType.BOOLEAN,
        param: {
            v1: {
                type: type.ParameterType.STRING,
                default: 'abc'
            },
            v2: {
                type: type.ParameterType.STRING,
                default: 'ABC'
            },
        },
        function: (args,util)=> ( args.v1 === args.v2 )
    },
```
